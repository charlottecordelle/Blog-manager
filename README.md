# Blog-Manager – React

**Stack :** React, TypeScript, gestion d’état (Hooks / Context / Redux), stockage local  

Projet frontend visant à créer un **mini-blog personnel** pour les étudiants et jeunes professionnels, permettant de **partager apprentissages, expériences et projets**.  
Ce projet met l’accent sur la création d’une application **typée, modulaire et persistante côté client**, démontrant la maîtrise de React et des bonnes pratiques frontend.

---

## Fonctionnalités

- Créer, afficher, modifier et supprimer des articles
- Organisation du contenu localement sans backend
- Gestion d’état moderne pour une expérience fluide
- Interface claire et responsive

---

## Installation

```bash
git clone <URL_DU_REPO>
cd day05/blog-manager
npm install
```
